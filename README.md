# Evonomics

A simple monetization-ready web app for consumer economics Q&A.

## What it does
- Accepts a consumer economics question.
- Produces a neutral, multi-perspective explanation.
- Shows recent trend graphs using World Bank data.
- Shows recent related news via GDELT.

## Run

```bash
python -m http.server 8000
```

Open: `http://localhost:8000`

## Monetization ideas
- Free tier: limited daily questions.
- Pro tier: saved dashboards + alerts.
- Enterprise: API and white-label analytics.
